# 🕒 Tamil Desktop Widget

A modern Windows desktop widget that displays the current time, English date, and Tamil calendar with a beautiful customizable interface.

---

## ✨ Features

- 🕒 Live Digital Clock
- 📅 English Date & Day
- 📜 Tamil Calendar Date
- 🎨 Solid Color Background
- 🌫 Transparent Background
- 🖼 Custom Background Image
- 🎨 Background Color Picker
- 🌈 English Text Color
- 🌈 Tamil Text Color
- 🔠 Custom Font Selection
- 📏 Adjustable Font Size
- 🔳 Adjustable Corner Radius
- 🌡 Adjustable Widget Opacity
- 📌 Always On Top
- 🚀 Start with Windows
- 📍 Save Widget Position
- 🖥 System Tray Support
- ⚙ Modern Settings Window

---

## 🛠 Technologies Used

- C#
- .NET 8
- Windows Presentation Foundation (WPF)
- XAML
- JSON Settings Storage
- Windows Registry
- Microsoft.Win32
- Windows Forms ColorDialog
